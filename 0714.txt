各機能を呼び出す（DBへ登録する等）部分を作成（作る前にgit pullで自身のディレクトリを最新に！）

		登録ボタンが押されたら、

		・mainメソッドは使用しない（Mainクラスでしか使用しない）
		・JFrameインスタンス削除
		　　・setDefaultClose...削除
		　　・setSize削除
		　　・setLocationRelativeTo削除
		・もともとmainメソッドに入っているものをコンストラクタ内に移動
	・（Javaファイル名：.java）
	


（！ファイルを作成するとき）
・担当ファイル以外は編集しない（特に同じファイル同じ行）
・作業が終わったらGitにあげる
　・先にgit pull origin mainを行い、他の人の変更を自分のローカルに取り込む
　・その後、
	git add <<編集したファイル名>>
	git commit -m "メッセージ"
	git push origin main
・（時間が余れば）自分が作成したファイルの先頭にコメントアウトでユーザ名だけ記述
	git pull origin main
	git commit -m "Creater Add"
